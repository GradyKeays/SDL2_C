# SDL2 Template (C + CMake)

A minimal SDL2 project in C using CMake on Windows, set up to support:

- SDL2 (core)
- SDL2_image (for PNG, JPG, etc.)
- SDL2_mixer (for audio playback)
- SDL2_ttf (for TrueType font rendering)
##  Build 

From the root of the project:

```bash
cd build
cmake ..
cmake --build .
